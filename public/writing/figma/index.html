Figma: an absolute beginner's guide

Figma is design software for producing layouts. It features vector design tools with options for embedding images, incorporating animations, and defining user flows. While many teams use it solely for its UI/UX capabilities and collaborative capabilities, it can be leveraged to build digital advertising for the web, social media, and eBooks. Figma effectively replaces Adobe Illustrator, InDesign, XD, and external collaboration/design review tools for digital design teams. However, it won't replace image editing, video, or 3D rendering tools.

This guide assumes you already have a Figma account and is the first in a three-part series to help users with muscle memory of the Adobe Suite interface make a smooth transition to using Figma.

The Interface

Figma runs in the browser, although there is a desktop app available. Both require access to the internet to run. Figma is set up to auto-save and will save your files locally if you lose internet connection. The interface is minimal, with a Toolbar at the top, Layers Panel at the left, and Design Panel at the right. Many UI elements are similar to those in the Creative Suite, with toggles for visibility on appearances, effects, and layers.

Toolbar

The toolbar features the Figma Menu, Cursor options, Frame options, Shape options, Pen/Pencil, Type, Move, and Comment at the left. Contextual tools appear in the center of the toolbar. Buttons to share, present, and view setting selection are available on the right. More details on the tools and menus are available in the following sections.

Canvas

You have a nearly infinite Canvas at the center of your window with 65,000 units in all directions. No more getting stuck in a corner as you add artboards. Also, unlike the Adobe Suite, you aren't forced to use an artboard at all. More on that in the Frames section. For more control over your raster effects, turn on Pixel Preview 1x or 2x to see how they will render outside of Figma. In addition to using Figma in light mode or dark mode, you can also change the color of the canvas. To do this, change the color with nothing selected.

Layers Panel

Like Illustrator, the Layers Panel displays all elements as individual layers. New layers are added at the top of the list. Layers can be grouped, nested, and reordered by dragging or keyboard shortcuts. Hover over Layers to display icons to toggle lock and visibility settings. Or select items and use the keyboard shortcuts. As with the Creative Suite, visibility and lock settings applied to groups also apply to children.

Design Panel

You can think of the information displayed in the Design Panel as everything you'd see in Adobe's properties panel, followed by context-specific panel options. We'll drill into this in more detail in the sections below.

Help

Quick links to helpful resources, including a complete list of all keyboard shortcuts, are available via a floating access button in the bottom right corner of the interface. Click here for a list of shortcuts displayed with their Adobe equivalent to help you work more efficiently. In addition, tool-specific shortcuts are listed within their respective sections below.

Frames and Slices

Frames function like artboards in Illustrator. You can add as many as you'd like and move them about the canvas as needed. You can draw your own, select from various pre-defined sizes, or create frames from existing objects/groups. Frames can be nested within Frames to create components such as a nav within a page. Frames can be manually resized using click and drag or selecting a new preset. You can also set Frames to shrink or grow with child objects. You can set Fill and Stroke effects directly on the frame - you don't need to include another element for styling.

Frames must be used to leverage Figma's specialized layout features, including Layout Grids, Constraints on Resize, Auto Layout, and Prototyping.

As in the Adobe Suite, Slices allow you to export selected sections of your design. In Figma, you can export Slices not located within a Frame. Frames or Slices can be exported in PNG, JPG, SVG, or PDF formats.

Working with Shapes

Working with shapes in Figma is similar to working in the Adobe Suite, although the keyboard shortcuts are different. Shapes are 100x100 default, and you can specify the width and height in the Design Panel once the primitive is on the canvas.

The shape controls vary little between the Adobe Suite and Figma. Position, dimensions, rotation, and corner radius controls are easily accessible via the top of the Design Panel. Select from pre-defined shapes or use the pen tool to build lines, polygons, or add and remove points as you would in Illustrator. Double click on a Shape to edit the BÃ©zier curves.

Circles and Arcs

Figma shines in how it handles circles/arc shapes. The tool lets you quickly create arcs, semi-circles, donuts, pie charts, and more via drag handles or by entering specific values.

Working with Text

Figma includes Google Fonts by default, provides for variable fonts, and allows you to use your local fonts. All the settings you'd expect from Adobe are available. Figma includes options to convert text to title case and reflow text when the container resizes.

List text

You can quickly create bulleted and numbered lists up to 5 levels deep via keyboard shortcuts, markdown syntax, or the Type Details Panel. Note that you cannot customize bullet characters; they will always appear round and black. They will, however, be bolded or take on the stroke settings of the type. Numbered lists alternate between Arabic, alphabetical, and roman numerals in that order and cannot be updated individually.

Fill, Stroke and other Visual Effects

The Fill section gives you quick access to hexadecimal codes and opacity settings. Click on the color chip to access more options. Here you'll see a color picker, eye dropper tool, the drop-down for various gradient effects, and fill with an image.

Selecting similar items

As with Illustrator, you can select objects with similar parameters by first selecting an object with properties you'd like to match and navigating to Figma Menu > Edit.

Editing Fills of Grouped Items

Selecting a group will bring up the Selections colors section of the Design Panel. From here, you can adjust colors without clicking on each group item individually.

Strokes

Color and opacity settings for strokes mirror working with fills. Figma allows you to set strokes for each edge independently from each other, allowing you to set a border-bottom effect on components easily.

Working with Images

Figma is a tool for layout design and has limited photo capabilities. Photos can be placed within shapes and set to fill, fit, crop or tile. Blend modes and Effects can be applied.

Blend Modes, Effects

The provided blending modes mirror those available in the Creative Suite. Both can be applied to Images, Layers, and Shapes. Some effects, however, work best with a combination of Shapes over Images.

Background Blur Setting

Create a blurred background by placing a shape on top of an image. On the Shape, reduce the opacity and set the Background Blur effect to allow a blurred background to appear.
