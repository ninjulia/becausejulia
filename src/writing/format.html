Style Text and More in Trello

Are you tired of just a block of standard text? Do you wish you could quickly bold text, and include italics, bullet points, numbered lists, or even images in your Descriptions or Comments?

Well, good news! Trello supports all that plus a few bonus features like embedding images and hexadecimal colors. If you've used Markdown syntax before, a lot of this will look familiar to you. If you're unfamiliar with Markdown, don't worry, a helpful table of Trello's supported Markdown features follows below.

Happy styling!

For Most Text Areas

Note: Formatting does not work in card titles. Not all syntax will display when using the Trello App.

Text Effect Syntax Keyboard Shortcut
Bold Text **text here** OR __text here__ ⌘/cmd + B
Italic Text *text here* OR _text here_ ⌘/cmd + I
Strikethrough ~~text here~~ ⌘/cmd + Shift + S
Linked Text [link text](link url) ⌘/cmd + K
Inline Code `code here` ⌘/cmd + Shift + M

Bonus Tip!

Hexadecimal color codes will display the color sample when entered.

For Descriptions and Comments Only

Header Type Syntax Alternate

Header 1

 #Header Text Header Text
===

Header 2

 ##Header Text Header Text

Header 3

 ###Header Text

Horizontal Rule

Be sure to put an extra line break before the rule or it will be formatted as a heading

Type This:

Some Text before rule
– – –
Some Text after rule

To Display This:

Bulleted and Numbered Lists

Be sure to include a line break before either list type or the formatting won't apply

Bulleted Lists

Type This:

- List Item 1
- List Item 2
- List Item 3
- List Item 4

To Display This:

Numbered Lists

Type This:

1. List Item 1
5. Numbers are auto-generated
6\. To preserve your numbering include slash after your number

To Display This:

Quote / Indented Text Styling

Type This:

>Innovation distinguishes
>between a leader and a
>follower.
>-Steve Jobs

To Display This:

Code Blocks

Type This:

```
<p>
     Hello World
<p>
```

But Wait! I don't *want* my text formatted. How can I avoid that?

Use a back slash after the characters: *\not italic text*\
