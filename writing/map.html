How to Map a Network Drive

Windows 10

1.

Open File Explorer.

2.

In the left pane, right click on This PC.

3.

From the drop-down, select Map network drive.

4.

Select any available letter from the Drive list.

5.

Select the folder to map by typing the file path into the Folder field or by clicking Browse to find the folder on the computer.

6.

Keep the Reconnect at sign-in box checked to connect to this folder whenever you log on to your computer.

7.

Click Finish.

Source: https://support.microsoft.com/en-us/help/4026635/windows-map-a-network-drive
